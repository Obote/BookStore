# 📖 [BookStore] <a name="about-project"></a>

**[BookStore]** is a project that uses a Single Page Application format to display items on a page. A user can switch to two pages wherein one displays the titles and authors of books, and also a form to add a new book.

## 🛠 Built With <a name="built-with"></a>

-HTML
-CSS
-JavaScript
-ReactJS
-Redux

## 💻 Getting Started <a name="getting-started"></a>

This project was bootstrapped with [Create React App](https://react.dev/learn/start-a-new-react-project).

### Prerequisites

In order to run this project you need:

Example command:

```sh
 git clone https://github.com/Obote/BookStore.git
```

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone https://github.com/Obote/BookStore.git
```

### Install

Install this project with:

```sh
  cd my-project
  npm install
```

### Usage

To run the project, execute the following command:

```sh
  npm start
```

### Run tests

To run tests, run the following command:

```sh
  npm test
```

You can deploy this project using:

```sh
  Deploy to any server of your choice
```
